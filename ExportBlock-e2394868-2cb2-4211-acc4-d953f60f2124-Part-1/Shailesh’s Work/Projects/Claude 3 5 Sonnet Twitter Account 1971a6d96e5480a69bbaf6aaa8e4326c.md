# Claude 3.5 Sonnet Twitter Account

Text: Gave Claude 3.5 Sonnet access to its own Twitter Account

## Context

Thought it would be interesting to see how Claude would react to tweets and what it thought about the state of the world. Gave access to a twitter account but since the Twitter Developer Account costs $200 (!) per month, I abandoned the project.

## Takeaways

I actually learned quite a bit from this project. I implemented the an integration with the Twitter API from scratch and I also learned how to use GitHub actions to trigger the posting of tweets by Claude.

Would have loved to carry this project but it can’t be done well without the Twitter API Premium Access.
